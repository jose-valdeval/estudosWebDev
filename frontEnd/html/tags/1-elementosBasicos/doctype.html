Doctype

Descrição

Tag usada pra especificar o tipo e versão de uma documentação

Fonte: Home Host

Exemplos:

No Codigo

		  <!DOCTYPE html>
		  <html>
		    <head>
		      <title>Olá Mundo!</title>
		    </head>
		    <body>
		      <h1>Primeiro título</h1>
		      <p>Primeiro parágrafo</p>
		    </body>
		  </html>

HTML Versão Mais Recente (HTML5)

			<!DOCTYPE html>

Usada para a versão mais recente do HTML

HTML Versão 4.1

			Strict:
			<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
			 "http://www.w3.org/TR/html4/strict.dtd">

			Transitional:
			<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
			 "http://www.w3.org/TR/html4/loose.dtd">

			Frameset:
			<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN"
			 "http://www.w3.org/TR/html4/frameset.dtd">

Strict: contém todos os elementos e atributos HTML mas não permite elementos de apresentação e nem o uso de Framesets

Transitional: contém todos os elementos e atributos mas não permite Framesets

Frameset: contém tudo o que o strict tem e o que não tem

XHTML Versão 1.0 - 1.1

			Strict:
			<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
			 "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

			Transitional:
			<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
			 "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

			Frameset:
			<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Frameset//EN"
			 "http://www.w3.org/TR/xhtml1/DTD/xhtml1-frameset.dtd">

			Simplificado?:
			<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN"
			 "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">

Todos (EXCETO XHTML1.1 "Simplificado") são parecidos com o HTML4.1 porém tem que ser escrito como XML bem formado

O XHTML1.1 "Simplificado" é parecido com o strict mas permite adicionar módulos.
