HTML

Descrição

Tag que conterá todo o codigo do documento, essa tag tambem é conhecida como "tag raiz" ou "tag root"

Exemplos:

HTML Simples:

		  <html>
		    <head>
		      <title<Olá Mundo!</title>
		    </head>
		    <body>
		      <h1<Primeiro titulo</h1>
		      <p<Primeiro paragrafo</p>
		    </body>
		  </html>
